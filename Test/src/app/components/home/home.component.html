Gestion des Bibliothèques du VAR

Utilité de cette application

Cette application gère la gestion des nos bibliothèques du VAR.

Vous trouverez toutes les informations necessaires(L'adresse,les contacts,les documents et les livres disponibles)

De plus,on peut ajouter tout le temps les nouveaux Bibliothèques,et on peut les modifier ou les supprimer.
